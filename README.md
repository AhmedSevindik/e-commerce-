## ✨ Features

- 🏠 Home screen with banner slider & categories
- 🛍️ Product list & detail screens
- 🔍 Search with filters
- 🌙 Dark / Light theme
- 🌍 Localization ready (EN/TR)
- ⚡ Shimmer loading states
- 📱 Clean, responsive UI

## 🏗️ Built With

- **State Management** – Riverpod
- **Navigation** – GoRouter
- **Architecture** – Feature-first clean architecture
